# gohyangon-donation

고향온 · 광주 서구 단독몰 **기부사업(EV) 페이지 화면설계서(SB)**.

전체 SB(`consthwi/gohyangon_sb`)에서 **기부사업 화면만 별도 추출**한 repo입니다.
필요한 기부사업 페이지만 **html.to.design 등 플러그인으로 깔끔하게 Figma import**하기 위해 분리 등록했습니다.

## 파일

- `all_pc.html` — 기부사업 PC 전 화면 세로 적층 (1회 import)
- `all_mo.html` — 기부사업 MO 전 화면 세로 적층 (1회 import)
- `index.html` — 허브(두 파일 링크 + 포함 페이지·기획 메모)

## 포함 페이지

- **EV-01 기부사업 목록/진입** (천원시리즈 전시 · 빈상태 포함)
- **EV-02 기부사업 상세** — 천원국시·천원택시·천원세탁(A 분리형) + 통합형(B안) 6종(세탁·국시·택시·기저귀·돌봄·중개보수)
- **SG-02 지정기부 상세(모금)** — e음 API 모금화면 (MO만)

> 구분: 페이지별 배경색. 회색 박스=이미지 영역, 점선 텍스트=BO 데이터(샘플).
> 원천 = `consthwi/gohyangon_sb` (단일 SB). 본 repo는 기부사업 페이지의 import 편의용 사본.
